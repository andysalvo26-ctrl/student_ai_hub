# Context Bundles Index

This directory contains drag-and-drop context bundles for drafting "AI by Smeal Major" pages. Each bundle is self-contained and ready for use in a fresh ChatGPT thread.

## How to Use

1. **First**: Drag `ai_by_smeal_major_system/CHATGPT_STARTER_CONTEXT.md` into ChatGPT
2. **Second**: Attach `AI by Smeal Major.pdf` (style witness)
3. **Third**: Drag the major bundle folder into ChatGPT
4. **Then**: Use prompts from `SECTION_PROMPTS.md` one section at a time

## Bundle Contents

Each major bundle contains:

- `README.md` - Usage instructions
- `MAJOR_CANON.md` - Canonical descriptive information (source of truth)
- `SHELL.md` - Section headings and structure
- `SECTION_PROMPTS.md` - Copy-paste prompts for Sections 2-5
- `BUNDLE_BOUNDARIES.md` - Quick reminder of constraints
- `NOTES.md` - Space for human notes during drafting
- `legacy/` - Old files moved here (STYLE_WITNESS.md, BOUNDARIES.md)
